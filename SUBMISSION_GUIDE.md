# Submission Guide for Scholars

## Where to submit

Applications are NOT submitted through GitHub. Submit through the Department's online intake (Google Form -> Drive), which keeps your personal data within the institutional environment. Contact phd@commerce.du.ac.in for the current intake link.

## Format

- Proposal / Research Protocol: A4, Arial or Times New Roman 11-12 pt, 1.5 spacing, numbered pages, max 15 pages excluding annexures.
- Supervisor-signed final proposal: a single locked PDF.
- Editable forms (DREC-01 to DREC-05): DOCX or PDF.
- Name each file Enrolment_Surname_FormNo.

## The complete packet (Guideline 4.2)

| # | Document | Requirement |
| --- | --- | --- |
| 1 | DREC Application Form (DREC-01) | Mandatory |
| 2 | Approved Synopsis / Proposal (supervisor-signed) | Mandatory |
| 3 | Research Protocol (per template) | Mandatory |
| 4 | Questionnaire / Interview Guide / Observation Schedule | If primary data |
| 5 | Participant Information Sheet (EN + regional) | If human participants |
| 6 | Informed Consent Form | If human participants |
| 7 | Organisational Access Permission Letter | If organisational data |
| 8 | Data Management Plan | Mandatory |
| 9 | AI Usage Declaration (DREC-03) | Mandatory |
| 10 | Risk Assessment Form (DREC-04) | Mandatory |
| 11 | Supervisor's Forwarding Letter | Mandatory |
| 12 | Conflict of Interest Declaration (DREC-05) | Mandatory |

Download the blank versions from the templates folder.
